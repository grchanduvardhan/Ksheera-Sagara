# 🐄 Ksheera-Sagara
## Dairy Profit / Loss Calculator

Ksheera-Sagara is an Android-based mobile application developed for dairy farmers to help them manage daily milk income, track farm expenses, and calculate overall profit or loss efficiently.

The application digitizes dairy farm accounting by providing a centralized platform where farmers can maintain financial records, monitor expenses, analyze cow-wise profitability, and generate monthly PDF reports.

# ✨ Features

- Daily Milk Income Tracking
- Expense Management System
- Automatic Profit / Loss Calculation
- Cow-wise Profit Analysis
- Dashboard Financial Summary
- Expense Breakdown Visualization
- Monthly PDF Report Export
- Profit-per-Liter Calculation
- Simple and User-Friendly Interface

---

# 📱 App Modules

## 🏠 Dashboard
- Displays total income, expenses, and net profit
- Shows profit/loss financial indicator
- Displays expense breakdown chart
- Monthly navigation support

## 🥛 Income Module
- Add daily milk entries
- Record:
  - Liters
  - Fat Percentage
  - SNF Percentage
  - Rate per Liter
- Auto-calculates total payment
- Morning/Evening session tracking

## 💸 Expense Module
- Track dairy farm expenses
- Expense categories:
  - Fodder
  - Medical
  - Labor
  - Other
- Edit and delete expense entries

## 🐄 Cow Analysis Module
- Add cow details
- Cow-wise profit calculation
- Performance tracking and analysis

## 📄 PDF Report Module
- Generate monthly financial reports
- Export PDF summary
- Share reports through other applications
